# Epigenetic clock reproduction (GSE40279)

## Project overview

This repository reproduces a methylation-based aging model from Illumina HumanMethylation450 beta values. The target is chronological age in a whole-blood cohort (GEO [GSE40279](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE40279)). The fitted learner is an **ElasticNetCV** regressor on a leakage-safe pipeline: univariate correlation is used only for dimension reduction, and the final model estimates **biological age** in the narrow sense of **DNA methylation–predicted age** for comparison with observed age (**epigenetic aging** as a supervised prediction problem).

## Scientific motivation

DNA methylation at specific CpG sites tracks age across tissues and has been widely used to construct **epigenetic clocks**. Re-implementing such a model from public data clarifies preprocessing choices, quantifies generalization on a held-out test set, and supports post hoc interpretation (probe stability, overlap with published clocks, functional enrichment). This project emphasizes **biological age prediction** from methylation while documenting steps that prevent train-test **information leakage**.

## Methodology

1. **Metadata parsing** — GEO series samples are read (e.g., via GEOparse); chronological age and covariates are extracted from SOFT characteristics.
2. **Beta matrix loading** — Illumina 450K beta values are read from the series supplementary data; probe IDs index columns and GSM sample IDs index rows.
3. **Alignment** — The beta matrix is intersected with labeled samples so each row of **X** matches a scalar age in **y** (same index set).
4. **Train/test split** — A single stratification-free split (80/20) is applied **before** any feature selection or scaling (`train_test_split`, fixed random state in the training notebook).
5. **Train-only correlation feature selection** — On the training partition only, probes are ranked by absolute Pearson correlation with age; the top *N* probes (default 5,000) are retained for the elastic net.
6. **Scaling** — `StandardScaler` is **fit** on the selected training features and **transform** is applied to train and test without refitting on the test set.
7. **ElasticNetCV** — Cross-validated elastic net regression selects `alpha` and `l1_ratio` on the training data; the refit model is evaluated on the hold-out test set.
8. **Evaluation** — Test-set MAE, RMSE, R², and Pearson correlation between predicted and observed age; optional repeated splits and permutation controls are recorded in `results/`.

## Leakage prevention

- **Feature selection** uses only training samples: correlations with age are computed from `X_train` and `y_train`; the test set does not influence which CpGs enter the model.
- **Scaling** uses training statistics exclusively: the scaler is fit on the training matrix and applied to the test matrix via `transform`, not `fit_transform`, on held-out data.
- **Test labels** are not used during preprocessing: no imputation, selection, or normalization step consults `y_test`.
- **Permutation sanity check** — Labels are shuffled on the training side within the same pipeline slot to verify that apparent performance is tied to true age labels (see `results/permutation_test_metrics.csv`). Shuffled-label controls show loss of signal relative to the real-label run.
- **Repository history** — Earlier notebook versions that risked leakage were removed during refactoring; the current training notebook documents a **leakage-safe** order of operations (split → select on train → scale on train → fit → evaluate on test).

## Model training pipeline

Execute `notebooks/02_model_training.ipynb` after `01_data_loading.ipynb`. The pipeline writes at minimum:

- `results/predictions.csv` — Observed vs. predicted ages on the test set.

## Biological interpretation

Run `notebooks/03_biological_interpretation.ipynb` after model training.

- **ELOVL2 / cg16867657** — In this repository’s fitted model, probe **cg16867657** (gene **ELOVL2**) carries the largest absolute coefficient among selected features and appears with selection frequency 1.0 across repeated split diagnostics, consistent with literature reporting ELOVL2-associated probes as strong age-associated signals on 450K data.
- **Overlap with the Horvath clock** — Non-zero model CpGs are compared to the Horvath clock probe list; enrichment against chance overlap is quantified (Fisher exact; see `results/enrichment_results.csv`).
- **Gene Ontology (g:Profiler)** — Among biological process terms, enriched categories include **cell adhesion** (including homophilic cell–cell adhesion), **developmental / multicellular organism development** processes, **nervous system development**, and **synaptic signaling** / trans-synaptic signaling (see `results/enrichment_results.csv` for identifiers, *p*-values, and query/term sizes).

## Reproducibility

1. Create a Python 3.11+ environment and install dependencies (see **Installation**).
2. Run notebooks **in order**:
   - `notebooks/01_data_loading.ipynb`
   - `notebooks/02_model_training.ipynb`
   - `notebooks/03_biological_interpretation.ipynb`
3. Use **Restart Kernel & Run All** on each notebook for a clean, deterministic execution path consistent with committed code.

Random seeds are fixed where specified in the notebooks (e.g., train/test split and repeated-split loops); full bitwise reproducibility additionally depends on library versions pinned in `requirements.txt`.

## Limitations

- **Single cohort / tissue** — Results apply to whole blood in GSE40279; generalization to other tissues or populations is not demonstrated here.
- **Correlation-based screening** — Univariate selection is simple and interpretable but ignores joint structure and can favor redundant probes.
- **Chronological age endpoint** — The model predicts reported age, not mortality or morbidity; “acceleration” here means residual from that regression, not necessarily clinical risk.
- **Batch and composition** — Plate and cell-composition effects are not fully modeled in the baseline pipeline; performance may partly reflect technical or confounding structure in the data.

## Future work

- Incorporate cell-type deconvolution or surrogate variables to separate biological aging from leukocyte composition.
- External validation on independent 450K or EPIC cohorts.
- Compare alternative selectors (group lasso, stability selection) and calibration under cohort shift.
